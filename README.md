# DIY 5000mAh Powerbank

A simple DIY powerbank built using a Li-Po battery, charging module, and boost converter.

## Components
- TP4056 Charging Module
- MT3608 Boost Converter
- 5000mAh Li-Po Battery
- USB Output Port
